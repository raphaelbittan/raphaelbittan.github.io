The Challenge

All the tricky technical details that bug us:
  1. Break The Lock: Redefined CI/CD Setup by migrating Vendor Lock Tools to Open Source Tools (from AWS CloudFront & Fargate to Jenkins) while keeping use of Cloud Provisioning (EC2, S3, EKS).
  2. Jenkins Scalp: Go through Pipeline, Determine Imperative/Declarative, Implement Parallel CI, Define Delivery, AquaSec, Slack Notification.
  3. Monitoring: Define correct dashboards with or without mixed datasources.
  4. Provisioning: Add automated SSL Certificate Authority, Let's Encrypt to the rescue.
